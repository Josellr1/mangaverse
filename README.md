# MangaVerse

> Tu visor de Manga, Manhwa y Manhua. Lee sin límites. 🌌

## 🌐 Demo en vivo
[Ver en GitHub Pages](https://tuusuario.github.io/mangaverse)

## ✨ Características
- Landing page premium con partículas animadas
- Lector web integrado (sin instalación)
- Búsqueda de miles de títulos
- Capítulos en 🇪🇸 Español, 🇺🇸 English y 🌐 Todos los idiomas
- Filtro de idioma
- Botones de descarga de la app móvil
- Avisos legales completos

## ⚖️ Aviso Legal
MangaVerse actúa únicamente como visor/reproductor. No almacenamos ni distribuimos contenido.  
Todo el contenido se obtiene de fuentes públicas (MangaDex API).  
Todos los derechos pertenecen a sus respectivos autores y grupos de scanlation.  
El uso de esta aplicación es responsabilidad exclusiva del usuario.

## 🛠️ Tecnologías
- HTML5, CSS3, JavaScript (Vanilla)
- [MangaDex Public API](https://api.mangadex.org)
- Font Awesome 6
- Google Fonts (Outfit)
